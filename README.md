# Matematică Vizuală 🧬📐

Un portofoliu de vizualizări matematice interactive și animații educaționale realizate cu **Manim** și **Vanilla JS**, conceput pentru a face matematica intuitivă prin reprezentări geometrice și dinamice.

---

## 📽️ Animații YouTube Shorts (Proiecte Manim)

### 1. Adunarea Fracțiilor (Ghid Vizual)
O explicație în două părți care demonstrează procesul de adunare a fracțiilor cu numitori diferiți prin divizarea și rearanjarea geometrică a pieselor.
*   **Partea 1:** Găsirea numitorului comun (Amplificarea). [🔗 Vizualizează pe YouTube](LINK_YOUTUBE_PART_1)
*   **Partea 2:** Adunarea și Simplificarea (10/24 -> 5/12). [🔗 Vizualizează pe YouTube](LINK_YOUTUBE_PART_2)
*   **Cod Sursă:** [adunare_fractii_vizual.py](scripts_manim/adunare_fractii_vizual.py)

### 2. Pătratul numerelor care se termină în 5
Demonstrație vizuală a formulei $(10n+5)^2 = 100n(n+1) + 25$ folosind rearanjări de arii.
*   [🔗 Vizualizează pe YouTube](LINK_YOUTUBE_SQUARE_5)
*   **Cod Sursă:** [square_ending_5_vertical.py](scripts_manim/square_ending_5_vertical.py)

### 3. Procent Swap (A% din B = B% din A)
Vizualizarea proprietății comutative a procentelor prin transformarea ariilor.
*   [🔗 Vizualizează pe YouTube](LINK_YOUTUBE_PERCENT_SWAP)
*   **Cod Sursă:** [procent_swap_vertical.py](scripts_manim/procent_swap_vertical.py)

### 4. Scumpiri Succesive
Analiza vizuală și algebrică a modului în care două creșteri procentuale consecutive afectează prețul final.
*   [🔗 Vizualizează pe YouTube](LINK_YOUTUBE_SCUMPIRE)
*   **Cod Sursă:** [scumpire_succesiva.py](scripts_manim/scumpire_succesiva.py)

## 🌐 Proiecte Interactive (Web)

### [Procente Vizual (Interactive Sliders)](procente.html)
O demonstrație interactivă pentru proprietatea comutativă a procentelor: $A\% \text{ din } B = B\% \text{ din } A$. Utilizatorii pot modifica valorile $A$ și $B$ și pot observa cum ariile dreptunghiurilor care reprezintă procentele rămân identice.

### [Pătrat Terminând în 5 (Web Tool)](patrat_5.html)
Versiunea interactivă a demonstrației geometrice pentru numerele de tipul $X5^2$.
