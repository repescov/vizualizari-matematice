# Matematică Vizuală

Un portofoliu de vizualizări matematice interactive, conceput pentru a face matematica mai ușor de înțeles prin reprezentări geometrice.

## Proiecte incluse:

### 1. Procente Vizual
O demonstrație interactivă pentru proprietatea comutativă a procentelor: $A\% \text{ din } B = B\% \text{ din } A$.
Utilizatorii pot modifica valorile $A$ și $B$ și pot observa cum ariile dreptunghiurilor care reprezintă procentele rămân identice, indiferent de forma dreptunghiului de bază.
